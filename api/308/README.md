### 直接使用 308 跳转 `/api/308`

1. 将 `/api/308` 目录下的函数部署至 `vercel` 或 `腾讯云函数` 

2. 自行选用数据库对接

3. 有备案域名的话更推荐 `腾讯云函数` 因为链接很干净

4. 而 `vercel` 的话，应该是要把函数文件放在 `api` 目录下才会生效，那么链接就变长了一点，不够优雅

5. 腾讯云不备案生成的链接就是个笑话
